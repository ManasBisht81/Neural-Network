🧠 Quantum Neural Network Visualization

An interactive 3D neural network simulation built using Three.js, featuring dynamic structures, glowing connections, and real-time user interaction.

🚀 Features

🌌 3D Neural Network Visualization

🎨 Multiple Color Themes (Purple, Sunset, Ocean)

⚡ Interactive Energy Pulses (Click anywhere)

🔄 Morphing Network Structures

🎛️ Adjustable Density Control

🧊 Pause / Resume Animation

🎥 Smooth Camera Controls (Orbit)

🛠️ Tech Stack

HTML5

CSS3 (Glassmorphism UI)

JavaScript (ES Modules)

Three.js

📂 Project Structure
project/
│── index.html
│── style.css
│── script.js
▶️ How to Run

⚠️ This project uses ES Modules, so it must run on a local server.

Option 1: Live Server (Recommended)

Open in VS Code

Install Live Server Extension

Right click → Open with Live Server

Option 2: Using Terminal
npx serve
🎮 Controls

🖱️ Click → Send energy pulse

🖐️ Drag → Rotate view

🎨 Theme buttons → Change color

🎚️ Slider → Adjust network density

🔄 Morph → Change structure

🧊 Freeze → Pause animation

🔁 Reset → Reset camera
